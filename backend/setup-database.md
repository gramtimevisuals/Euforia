# Quick Setup

1. **Get Supabase URL**: Go to supabase.com → Create project → Copy URL
2. **Update .env**: Paste your Supabase URL
3. **Run**: `npm start` - Database will auto-connect

## Your .env should look like:
```
SUPABASE_URL=https://your-project-id.supabase.co
SUPABASE_ANON_KEY=your_anon_key
JWT_SECRET=any_secret_string
```

That's it! The app will connect automatically.
